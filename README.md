# xmind-test

## 所需环境：Windows/macOS/linux，node

## clone 项目之后，首先需要安装项目所需依赖，在项目根目录执行

```
npm install
```

### 依赖安装完成后，可通过如下指令运行项目，然后打开提示的地址即可看到项目

```
npm run serve
```

### 另外：也可通过如下指令，得到打包之后的文件（在/dist 文件夹中），然后将这些文件放到任意 web 环境中即可访问

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
